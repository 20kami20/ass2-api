### Setup

2. Create `.env` with API keys

### APIs Used
- RandomUser API
- Countrylayer API
- ExchangeRate API
- News API

### Description
The application fetches a random user, retrieves country details, currency exchange rates,
and displays related news headlines. All API calls are handled on the server side.
